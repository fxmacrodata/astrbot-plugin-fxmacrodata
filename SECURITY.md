# Security policy

This repository contains only the FXMacroData AstrBot client plugin. It must
never contain API keys, OAuth tokens, encryption keys, customer data,
FXMacroData backend code, ingestion/source logic, proprietary datasets, or
commercially sensitive implementation details.

Do not place credentials or authenticated URLs in GitHub issues, pull
requests, screenshots, test fixtures, logs, or configuration examples.

If you identify a security issue, use GitHub private vulnerability reporting
when it is enabled for this repository. Otherwise, open a minimal issue that
does not disclose an exploit, credential, or customer information and request
a private reporting channel.
